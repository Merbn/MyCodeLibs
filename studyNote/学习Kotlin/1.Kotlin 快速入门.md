# Java 和 Kotlin 语言对比
## 打印日志
+ Java
```
System.out.print("amit Sheckeer");</br>
System.out.println("换行打印");
```
+ Kotlin
```
print("普通打印");</br>
println("换行打印");
```
## 常量与变量
+ Java
```
String name ="merbng";</br>
final String name ="merbng;
```
+ Kotlin
```
var name ="merbng"</br>
val name ="merbng"
```
## null声明
+ Java
```
String otherName ;</br>
otherName = null;
```
+ Kotlin
```
var otherName : String?</br>
otherName = null
```
##空判断
+ Java

```
if(text != null){
    int length =text.length();
}
```
+ Kotlin

```
text?.let{
    val length =text.length
}
```
